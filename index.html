1Programación web

La programación de los sitios web es una de las disciplinas dentro del mundo de Internet que más se ha desarrollado y no deja de sorprender día a día con las posibilidades que abre y genera, ya que no sólo consigue satisfacer necesidades que se generan, sino que sin la generación de necesidades ofrecen servicios a los usuarios que éstos no habían imaginado.

2Html, Css, JavaScript

Una descripción general:

  • HTML proporciona la estructura básica de los sitios, que es mejorada y modificada por otras tecnologías como CSS y JavaScript.
  • CSS se utiliza para controlar la presentación, el formato y el diseño.
  • JavaScript se utiliza para controlar el comportamiento de diferentes elementos.

    • 3Diseño web

    El diseño web es un área enfocada en el desarrollo de interfaces digitales, como el diseño de sitios y aplicaciones para web. Para ello, los diseñadores web crean las páginas utilizando lenguajes de marcado como HTML.
